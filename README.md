# Angels Calendar 2026

A React-based editorial calendar for the 2026 Angels baseball season.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open in browser: http://localhost:5173

## Deploy to Vercel

1. Push this repo to GitHub
2. Connect to Vercel
3. Deploy!

## Tech Stack

- React 18
- Vite
- Tailwind CSS v4
- Lucide React Icons
